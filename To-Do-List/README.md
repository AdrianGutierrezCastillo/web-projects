# ✅ To-Do List

Una aplicación web sencilla para gestionar tareas diarias de forma rápida e intuitiva.

## 📖 Descripción

Este proyecto permite crear, visualizar y eliminar tareas, ofreciendo una interfaz simple y fácil de usar. Fue desarrollado como práctica para fortalecer habilidades en desarrollo web y manipulación del DOM.

## ✨ Características

- ➕ Agregar nuevas tareas.
- 🗑️ Eliminar tareas.
- 📱 Diseño responsive.
- ⚡ Interfaz sencilla e intuitiva.

## 🛠️ Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
