# 🛒 E-Commerce

Aplicación web de comercio electrónico con autenticación de usuarios, catálogo de productos, carrito de compras y panel de administración.

## 🛠️ Tecnologías

- HTML5
- CSS3
- JavaScript
- LocalStorage
